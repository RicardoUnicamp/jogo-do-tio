git clone -> Clona o projeto no git para sua máquina.
Untracked -> Funcionalidade que ainda não foi adicionada a um pacote de mudanças (Não foi commitada).
Tracked -> Funcionalidade que foi adicionada ao projeto (Commitada).

git status -> ciclo de estado dos meus arquivos
git add ->mostra para o git quais arquivos ele deve verificar
